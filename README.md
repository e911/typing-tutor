# Typing Tutor

Typing tutor is a simple game to practice and enhance your typing skills.

## Installation

Clone the project: 

SSH: 
```bash
    git clone git@github.com:e911/typing-tutor.git
```

HTTPS:
```bash
    git clone https://github.com/e911/typing-tutor.git
```

Install all the dependencies:
```bash
    cd typing-tutor
    npm install
```

Run the project:
```bash
    npm run start
```

####Enjoy.
